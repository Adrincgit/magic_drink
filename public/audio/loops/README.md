# Audio Loops — Magic Drink Flavors

Cada sabor tiene su propio loop musical.  
Misma melodía base, diferente tono/instrumento por sabor.

## Archivos esperados (8-15 segundos, loop seamless):

| Archivo | Sabor | Tono sugerido |
|---------|-------|---------------|
| `loop_original.mp3` | Magic Original | Base melódica púrpura, synth pads cálidos |
| `loop_kiwi.mp3` | Witchy Kiwi | Versión tropical, marimbas + plucks frescos |
| `loop_sparkle.mp3` | Sparkle Soda | Celestial, bells, arpeggios brillantes |
| `loop_banana.mp3` | Banana Drama | Energético, brass stabs tropicales |
| `loop_bubble.mp3` | Bubble Tape | Retro/bubblegum, 8-bit + lo-fi vibes |
| `loop_dragon.mp3` | Dragon Grape | Intenso, bass profundo, dramático |

## Formato recomendado:
- MP3 a 192kbps o superior
- 44.1kHz estéreo
- 8-15 segundos de duración
- Loop point limpio (sin clicks al repetir)
